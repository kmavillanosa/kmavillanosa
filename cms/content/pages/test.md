What IPPY does
Backs up iPhone media files directly to a Windows PC
Uses Flask (Python) as a lightweight local server
Uses iPhone Shortcuts for easy automation

Fully self-hosted, no cloud dependency

Prerequisites:
Python installed on your PC
Clone the repository:
<https://lnkd.in/g34FY7nb>

iPhone with the Shortcuts app installed:
<https://lnkd.in/gMhr3TDA>

Phone and PC must be on the same network during setup
Further setup instructions are available in the repository documentation.
